# try catch OX 문제

# try-catch-finally 구문

- 고급 예외 처리는 try 키워드, catch 키워드, finally 키워드로 예외를 처리하는 방법
- 이를 try catch finally 구문이라고 부름

## 기본 구조

```csharp
try
{
    // 예외가 발생하면 (If an exception occurs)
}
catch (Exception exception)
{
    // 여기서 처리합니다. (Handle it here.)
}
finally
{
    // 여기는 무조건 실행합니다. (This is always executed.)
}
```

## 구문 생략 가능

다음과 같이 catch 구문 또는 finally 구문이 필요 없다면, 해당 부분을 생략하고 사용할 수 있음

### try-catch 구조

```csharp
try
{
    // 예외가 발생하면
}
catch (Exception exception)
{
    // 여기서 처리합니다.
}
```

### try-finally 구조

```csharp
try
{
    // 예외가 발생하면 그냥 넘어갑니다.
}
finally
{
    // 여기는 무조건 실행합니다.
}
```
