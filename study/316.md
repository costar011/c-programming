# 2문제 냈음

## 01 다음 문장이 맞다면 O, 틀리다면 X 하시오.

1. 메서드에서 아무것도 리턴하지 않을 때는 반환형(리턴 자료형)을 Null로 입력한다.

   - **X**: void로 입력한다.

2. 메서드는 여러 개의 값을 매개변수로 받을 수도 있고 반환할 수도 있다.

   - **X**: 매개변수는 여러 개를 받을 수 있지만, 반환하는 값은 최대 1개이다.

3. 같은 이름으로 매개변수가 다른 메서드를 여러 개 만드는 것을 오버라이드라고 부른다.

   - **X**: 오버로딩(Overloading)이다.

4. private 접근 제한자가 붙은 변수와 메서드는 해당 클래스 내부뿐만 아니라 외부에서도 접근할 수 없다.

   - **X**: 해당 클래스 내부에서는 사용할 수 없다.

5. 생성자에는 public 접근 제한자만 붙일 수 있다. private 접근 제한자를 붙이면 오류가 발생한다.

   - **X**: 일반적으로 생성자에는 public 접근 제한자를 붙이며, private 접근 제한자도 붙일 수는 있다.

6. C# 개발자들은 일반적으로 클래스 변수의 이름은 대문자로 시작하게 하고, 속성 이름은 소문자로 시작하게 해서 구분한다.

   - **X**: 클래스 변수와 속성 이름 모두 대문자로 시작하게 한다.

7. 윈도 폼의 이벤트 메서드에서 매개변수 sender는 이벤트를 발생시킨 객체를 의미한다.
   - **O**

## 02 다음 빈칸을 채우시오.

1. 클래스 메서드는 ( **static** ) 키워드를 사용하여 선언한다.

2. 소멸자는 클래스의 이름 앞에 ( **~** ) 기호를 붙여 선언한다.

3. 접근 제한자 중에서 모든 곳에서 접근 가능한 것은 ( **public** )이다.

## 03 다음 중 생성자에 대한 설명으로 옳지 않은 것을 고르시오.

① 생성자 이름은 클래스 이름과 같다.

② 생성자는 반환과 관련된 선언을 하지 않는다.

③ 생성자는 인스턴스가 생성될 때 자동으로 호출되는 메서드이다.

④ 클래스는 반드시 개발자가 직접 정의해야 한다.

**답: ④**

```csharp
class Program
{
```

## 08 다음 코드의 실행 결과를 예측하시오.

```csharp
class A
{
    public A()
    {
        Console.WriteLine("A의 생성자");
    }
    ~A()
    {
        Console.WriteLine("A의 소멸자");
    }
}

class B : A
{
    public B()
    {
        Console.WriteLine("B의 생성자");
    }
    ~B()
    {
        Console.WriteLine("B의 소멸자");
    }
}

class Program
{
    static void Main(string[] args)
    {
        {
            new B();
        }
    }
}
```

**실행 결과:**

```
A의 생성자
B의 생성자
```

## 11 다음 코드의 실행 결과를 예측하시오.

```csharp
class Program
{
    class Test
    {
        public int value = 10;
    }

    static void Main(string[] args)
    {
        Test test = new Test();
        int a = test.value;
        test.value = 20;
        Console.WriteLine(test.value + " " + a);
    }
}
```

**실행 결과:**

```
20 10
```

**설명:**
변수 test는 객체라서 참조를 저장하지만, test.value 자체는 기본 자료형이라 값을 저장한다. 이를 복사했기 때문에 a는 값 복사한다
