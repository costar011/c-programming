# 1번 문제, 5번 (8 보기 문제), 9~12번 응용 문제

## 01 다음 빈칸을 채우시오.

1. 클래스 간에 부모-자식 관계를 만드는 것을 ( **상속** )이라고 부른다.

2. C#에서 상속을 구현할 때는 ( **:** ) 기호를 사용한다.

3. 부모 클래스에서 ( **public** )과 ( **protected** ) 접근 제한자가 붙은 멤버는 자식 클래스에서 접근할 수 있다.

4. 부모 클래스에서 ( **private** ) 접근 제한자가 붙은 멤버는 자식 클래스에서 접근할 수 없다.

5. 자식 클래스에서 부모 클래스를 참조할 때는 ( **base** ) 키워드를 사용한다.

6. 자식 클래스에서 부모 클래스의 메서드를 숨기려면 ( **new** ) 키워드를 사용한다. ( **override** ) 키워드를 사용하면 부모 클래스의 메서드를 덮어쓸 수 있다.

7. 특정 범위의 변수가 외부 범위의 변수와 같은 이름을 가져서 외부 범위의 변수를 숨기는 것을 ( **섀도잉** )이라고 부른다.

8. 클래스의 상속을 막을 때는 ( **sealed** ) 키워드를 사용한다.

9. 메서드의 오버라이드를 막을 때는 ( **sealed** ) 키워드를 사용한다.

10. ( **abstract** ) 키워드가 붙은 클래스는 인스턴스를 만들 수 없다.

11. 자식 클래스에서 부모 클래스의 메서드를 덮어쓸 때는 부모 메서드에 ( **virtual** ) 키워드를 붙이고, 자식 메서드에 ( **override** ) 키워드를 붙인다.

## 05 다음 코드에서 예외가 발생하는 부분을 고르시오.

```csharp
class Unit {}
class Tank : Unit { }

class Program
{
    static void Main(string[] args)
    {
        Unit unit = new Unit();
        Tank tank = new Tank();

        /* 1 */ Unit a = (Unit)unit;
        /* 2 */ Unit b = (Unit)tank;
        /* 3 */ Tank c = (Tank)unit;
        /* 4 */ Tank d = (Tank)tank;
    }
}
```

**답: 3**

## 08 as 키워드와 is 키워드의 차이점을 설명하시오.

**답:**

`is`는 객체가 특정 형식인지 확인해서 `true` 또는 `false`를 반환한다. `as`는 객체를 특정 형식으로 변환하려고 시도하는데, 실패 시 예외를 발생시키는 대신 `null`을 반환한다.

## 09 다음 코드의 실행 결과를 예측하시오.

```csharp
class Parent
{
    public int Question() { return 10; }
}

class Child : Parent
{
    public int Question() { return 20; }
}

class Program
{
    static void Main(string[] args)
    {
        Child child = new Child();
        Console.WriteLine(child.Question());
    }
}
```

**실행 결과:**

```
20
```
