# 인덱스 코드 (빈칸 문제)

```csharp
class Products
{
    public int this[int i]  // 인덱서를 선언합니다.
    {
        // Products products = new Product();
        // products[i] 할 때에 호출
        get { return i; }
        // products[i] = 10 할 때에 호출
        set { Console.WriteLine(i + "번째 상품 설정"); }
    }
}
```

## 설명

- **인덱서 선언**: `public int this[int i]` - `this` 키워드를 사용하여 인덱서를 선언합니다.
- **get 접근자**: `products[i]`로 값을 읽을 때 호출됩니다.
- **set 접근자**: `products[i] = 10`으로 값을 설정할 때 호출됩니다.
