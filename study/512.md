# 7,8번 직접 쓰기 문제

## 07 다음 델리게이터를 이용하여 두 개의 정수를 더하는 코드를 작성하시오.

```csharp
delegate int AddDelegate(int x, int y);
```

**답:**

```csharp
AddDelegate add = (x, y) => x + y;
Console.WriteLine(add(3, 4)); // 출력: 7
```

## 08 다음 델리게이터를 이용하여 문자열을 출력하는 코드를 작성하시오.

```csharp
delegate void PrintDelegate(string message);
```

**답:**

```csharp
PrintDelegate print = delegate(string message) {
    Console.WriteLine(message);
};
print("Hello, World!"); // 출력: Hello, World!
```
